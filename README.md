## Em desenvolvimento...

***Instalar e ativar o Virtualenv:***

~~~shell
    pip install virtualenv
~~~~
***OU***
~~~shell
    virtualenv --system-site-packages -p python ./venv
~~~~
***Ativar virtualenv***
~~~shell
    venv/scripts/activate
~~~~
